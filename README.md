# EcoTech Solutions Website

This is a static website for EcoTech Solutions, a South African eco-friendly technology and renewable energy consulting business.

## Pages

- `index.html` — homepage with company overview, services preview, and brand messaging.
- `about.html` — company story, mission, vision, values, and target audiences.
- `services.html` — detailed service offerings with pricing examples.
- `blog.html` — blog-style content with sustainability guides and energy tips.
- `contact.html` — contact form, local South African address, phone number, and embedded map.

## Styling

- `Style.css` contains the global design system, including colors, typography, layout, responsive behavior, and background image.
- The site uses a subtle full-page background image and a small watermark logo in the header.

## Local preview

To preview the website locally, open the HTML files directly in your browser or use a simple local server.

### Option 1: Open directly

1. Open the `EcoTech` folder.
2. Double-click `index.html`.
3. Use the navigation links to visit other pages.

### Option 2: Serve with a local web server

If you have Python installed, run this from the `EcoTech` folder:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000/index.html
```

## Notes

- Contact information is set for a South African office in Sandton, Johannesburg.
- Service pricing is shown as example starting costs.
- The site is designed for responsive display on desktop and mobile.
